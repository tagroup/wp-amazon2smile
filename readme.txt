WordPress Plugin that changes Amazon Affiliate links to Smile Amazon Affiliate Links

== Description ==

Looks for normal Amazon affiliate links and switches the www with smile so the affiliate can still get the referral and the customer can still donate to their cause.

More information http://thomasloughlin.com/amazon-smile-amazons-affiliate-program/ 

The plugin itself is doing a regex to find an affiliate link and then replacing the www. Very small.

It's pretty simple to run, just drop it in the plugin directory and activate that's all it takes.

== Installation ==

1. Upload the complete directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Test - You assume all risk in changing urls.

== Frequently Asked Questions ==

= Can I use this to change shortened Affiliate affiliate links to smile.amazon.com affiliate links? =

No, you will want to download the other plugin mentioned at http://thomasloughlin.com/amazon-smile-amazons-affiliate-program/
